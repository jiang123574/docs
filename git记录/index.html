工作区及暂存区
工作区就是存有.git 文件夹的父文件夹，暂存区是在.git 文件夹里的区域。
我们执行的 git add [.]就是把文件从工作区添加到了暂存区(stage)中，而 git commit 就是把暂存区里的东西提交到当前的本地分支比如 test 分支，工作区就变干净了。
